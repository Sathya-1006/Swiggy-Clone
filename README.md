# 🍔 Swiggy Clone
A simple **Swiggy Clone** built using **HTML** and **CSS**.
This project replicates the basic landing page and UI structure of the Swiggy food delivery platform.

---

## 🚀 Live Demo

🔗 [Click here to view](https://swiggyclone-omega.vercel.app/)

---

## 🛠️ Tech Stack

* **HTML5** – for structure
* **CSS3** – for styling and layout

---

## ✨ Features

* Swiggy-inspired landing page design
* Responsive layout (basic)
* Clean and minimal UI

---

## 💻 How to Run Locally

1. Clone this repository:

   ```bash
   git clone https://github.com/<your-username>/<your-repo>.git
   ```
2. Navigate to the project folder:

   ```bash
   cd <your-repo>
   ```
3. Open `Swiggy.html` in your browser.
